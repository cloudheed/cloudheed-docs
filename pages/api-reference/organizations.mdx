# Organization Endpoints

API reference for organization management endpoints.

import { Callout } from 'nextra/components'

## Create Organization

<span className="method-badge method-post">POST</span> `/v1/organizations`

### Request Body

```json
{
  "name": "My Company",
  "type": "business"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Organization name |
| `type` | string | Yes | `personal` or `business` |

### Response `201 Created`

```json
{
  "id": "org-abc123",
  "name": "My Company",
  "slug": "my-company",
  "type": "business",
  "billing_email": "user@example.com",
  "created_at": "2026-03-17T10:00:00Z",
  "updated_at": "2026-03-17T10:00:00Z"
}
```

## Delete Organization

<span className="method-badge method-delete">DELETE</span> `/v1/organizations/{id}`

<Callout type="warning">
  You must delete all databases and settle all invoices before deleting an organization.
</Callout>

## List Members

<span className="method-badge method-get">GET</span> `/v1/organizations/{id}/members`

<Callout type="info">
  Team features are only available for business organizations.
</Callout>

### Response

```json
{
  "members": [
    {
      "id": "member-abc123",
      "user_id": "user-123",
      "role": "owner",
      "joined_at": "2026-03-17T10:00:00Z",
      "user": {
        "id": "user-123",
        "email": "owner@example.com",
        "name": "John Doe"
      }
    }
  ]
}
```

### Roles

| Role | Description |
|------|-------------|
| `owner` | Full access, can delete org |
| `admin` | Manage members and databases |
| `member` | View and use databases |

## Invite Member

<span className="method-badge method-post">POST</span> `/v1/organizations/{id}/invites`

### Request Body

```json
{
  "email": "newmember@example.com",
  "role": "member"
}
```

### Response `201 Created`

```json
{
  "id": "invite-abc123",
  "email": "newmember@example.com",
  "role": "member",
  "expires_at": "2026-03-24T10:00:00Z",
  "created_at": "2026-03-17T10:00:00Z"
}
```

---
